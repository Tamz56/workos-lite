# WorkOS-Lite (ArborDesk)

Local-first personal Work OS (Tasks + Docs) built with Next.js.

## Quickstart
```bash
npm install
npm run dev
```

## Scripts
- `npm run lint` - Check for linting errors
- `npm run build` - Build for production

## Key routes
- `/dashboard` - Overview & hygiene checks
- `/inbox` - Task inbox
- `/today` - Daily planner
- `/planner` - Long-term planning
- `/docs` - Markdown docs management
